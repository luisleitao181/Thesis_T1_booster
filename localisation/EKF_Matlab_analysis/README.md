# EKF MATLAB Analysis – Booster T1

This folder contains the Extended Kalman Filter (EKF) implementation and experiments used for localization analysis.

## Structure

### /main_EKF_sim
Core MATLAB simulation code implementing the EKF model.

### /experiments
Basic validation experiments used for the related article.

### DH_simplified.csv
Denavit–Hartenberg parameter table used for kinematic modeling.
Includes reference configuration used in simulations.
